# SYSU VLSI Design Labs

This repository contains coursework for SYSU DCS3024 VLSI Design.

## Lab 01: Hypergraph Bipartitioning

Lab 01 implements a 2-way hypergraph partitioner for the VLSI-FPGA Lab 1 assignment. Given an `.hgr` netlist, the program outputs two balanced partitions and reports the cut cost.

Layout:

- `lab/01/src/`: C++ source code based on the official starter framework.
- `lab/01/data/`: local ISPD `.hgr` input data, ignored by Git.
- `lab/01/build/`: compiled binary, object files, logs, and partition outputs, ignored by Git.
- `lab/01/report/`: Typst report source and `result.csv` generated by `run.sh`.

Run all benchmarks:

```bash
cd lab/01
bash run.sh
```

Build the report:

```bash
cd lab/01/report
typst compile main.typ <report-name>.pdf
```
